# Horror Mine - Escape or Die

A thrilling horror game where you must escape from a dark, dangerous mine or face the consequences.

## Overview

Horror Mine is an interactive game built with HTML5 Canvas and JavaScript. Navigate through treacherous environments, survive encounters, and find your way out of the mine before it's too late.

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required

### How to Play

1. Open `index.html` in your web browser
2. Use the controls to navigate through the mine
3. Avoid hazards and threats
4. Find the exit to escape and win

## Features

- Interactive Canvas-based graphics
- Immersive horror atmosphere with sound design
- Score and progress tracking
- Crosshair-based targeting system
- Multiple game states (playing, win, lose)
- Responsive controls and gameplay

## Controls

- **Mouse cursor**: Aim with the crosshair
- **Mouse click**: Interact/attack
- **Arrow keys or WASD**: Move around
- Check in-game UI for additional controls

## Game States

- **Playing**: Navigate and survive in the mine
- **Win**: Successfully escape the mine
- **Lose**: Fail to escape and the game ends

## Technical Details

- Built with vanilla HTML5, CSS, and JavaScript
- Uses Canvas API for rendering
- Responsive design for different screen sizes
- Per-pixel perfect crosshair display
